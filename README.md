# Study Motivation App-Prototype

A study productivity app prototype designed in Figma.
The goal of the app is to help students stay focused, complete tasks, and stay motivated while studying.

Users can create to-do lists, use focus timers, and earn rewards for completing tasks.

# Features:
# To-Do List
Create and manage study tasks, track completed tasks
# Focus Timer 
Set custom focus sessions, helps users stay concentrated while studying
# Reward System
Earn coins for completed tasks or focus sessions, coins can be spent in the in-app shop
# Shop
Unlock themes, buy outfits for the character
# Statistics
Track study minutes, visualize productivity over time
# Gamification
Level system, daily and special quests, progress rewards
